# @skt-t1-byungi/array-includes
`includes()` for old browser, in typescript.

## Example
```js
includes([1, 2, 3], 2)
// => true

includes([1, 2, 3], 4)
// => false
```

## includes(array, search, from = 0)
### array
Target array.

### search
The element to search for.

### from
The position in this array at which to begin.
