# @skt-t1-byungi/array-map
`map()` for old browser, in typescript.

## Example
```js
map([1, 2, 3], x => x * 2)
// => [2, 4, 6]
```

## map(array, mapper, [thisArg])
### array
Target array.

### mapper
Function that produces an element of the new Array.

### thisArg
Value to use as this when executing callback.
