# @skt-t1-byungi/array-flat
`flat()` for old browser, in typescript.

## Example
```js
flat([1, 2, [3, 4]])
// => [1, 2, 3, 4]
```

## flat(array, depth = 1)
### array
Target array.

### depth
The depth level specifying how deep a nested array structure should be flattened. Defaults to 1.
